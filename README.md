# Example Django REST

Example Django REST sandbox project/ 

## Installation

Use the package manager [pipenv](https://github.com/pypa/pipenv) to install project.

```bash
$ pipenv install --dev
```

## Usage

Setup `.env` configration file.

```bash
$ pipenv run make_env
```

This will create `.env` file in root from `.env.example` file.
